# 💎 Inayat Royale — Where Every Sparkle Tells a Royal Story

**Inayat Royale** is a premium, luxury e-commerce platform designed for high-end handcrafted jewellery. Built with a focus on elegance, trust, and a seamless user experience, this platform bridges the gap between traditional craftsmanship and modern digital commerce.

![Hero Banner](Apps/web/public/hero_banner.jpg)

---

## ✨ Key Features

### 👑 Premium Customer Experience
- **Luxury Aesthetic**: A sophisticated dark-themed design (#0a0a0a) with gold accents that reflects the brand's royal heritage.
- **Dynamic Collections**: Browse curated collections of Gold, Silver, and Bridal jewellery with high-resolution imagery and transparent pricing.
- **WhatsApp Integration**: Seamless one-click inquiry flow that connects customers directly with sales experts for personalized consultations.
- **Responsive Design**: Flawless experience across Mobile, Tablet, and Desktop devices.

### 🛡️ Powerful Admin Dashboard
- **Product Management**: Easily add, edit, and remove products with image upload support and category organization.
- **Lead Tracking**: Centralized "Contact Submissions" panel to view and manage customer enquiries in real-time.
- **Secure Access**: Protected administrative routes with secure authentication and password management.
- **Database Control**: Powered by **PocketBase**, providing a lightweight, high-performance backend with instant API responses.

### 🚀 Performance & SEO
- **Vite-Powered**: Ultra-fast development and build times.
- **SEO Optimized**: Fully implemented Meta tags, Open Graph (OG) support, and JSON-LD structured data for superior search engine visibility.
- **Smooth Animations**: Powered by **Framer Motion** for a fluid, premium feel.

## 🛠️ Tech Stack

- **Frontend**: [React 18](https://reactjs.org/), [Vite](https://vitejs.dev/), [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn/UI](https://ui.shadcn.com/), [Lucide Icons](https://lucide.dev/)
- **State & Logic**: [React Hook Form](https://react-hook-form.com/), [Zod](https://zod.dev/)
- **Backend**: [PocketBase](https://pocketbase.io/) (Go-based BaaS)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AmanSharma0812/Inayat-Royale.git
   cd Inayat-Royale
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the PocketBase backend**:
   - Navigate to `Apps/pocketbase`
   - Run `./pocketbase.exe serve --http=127.0.0.1:8091`

4. **Run the frontend**:
   ```bash
   npm run dev
   ```
   *The site will be available at `http://localhost:3000`*
